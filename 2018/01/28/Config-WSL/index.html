1. Windows SubSystem Linux 介绍

个人Linux属于一帮爱折腾家伙的玩物，之前想玩一般是在虚拟机里玩，而Windows 在 Windows 10 里直接引入了这么一个子系统，可以说这还是相当给力的。

有很多开发者都喜欢用 Mac，包括我，他们给出的理由是 OSX 对开发者友好，而 WSL 的引入恰好可以弥补这一点，为 Windows 系统争取更多的使用者。

就我本人而言，还是喜欢看到这种竞争的。

1.1 WSL 安装

现在我知道有两种方式：

  1. 在 Windows Store 中安装
  2. 使用 LxRunOffline
